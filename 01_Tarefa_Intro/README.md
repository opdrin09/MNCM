# Tarefa 1: Introdução (Julia)

Esta pasta contém os scripts introdutórios da disciplina, desenvolvidos em Julia e Jupyter Notebook.

## Arquivos
- `tarefa1.jl`: Script Julia com a resolução base.
- `executar_tarefa_1.ipynb`: Notebook Jupyter documentando a execução e resultados.

## Como Executar
Para rodar estes arquivos, é necessário ter o ambiente Julia configurado (kernel IJulia para o notebook).

```bash
julia tarefa1.jl
# ou abra o notebook
jupyter notebook executar_tarefa_1.ipynb
```
